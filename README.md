這是一個基於 ESP8266/ESP32 微控制器的 空氣品質與環境監測系統，透過 MQ-135 空氣品質感測器 和 DHT11 溫濕度感測器 收集數據，並顯示在 SSD1306 OLED 顯示屏 上。

📌 功能介紹

讀取 MQ-135 感測器 的 空氣品質數據（Good/Bad 指標）。

讀取 DHT11 感測器 的 溫度與濕度 數據。

在 OLED 顯示器 上顯示 空氣品質、溫度與濕度資訊。

每 5 秒 更新數據。
